# HealthMate Plus

HealthMate Plus is a frontend wellness dashboard for tracking daily metrics, goal progress, reminders, and health insights.

## Features
- Log health data (steps, water, sleep, calories)
- Set daily/weekly goals with progress bars
- Simulate tracker sync
- Personalized tips based on logged data and goals
- Reminder scheduling with in-app notifications
- Export all data as JSON
- Data persistence via browser localStorage

## Run
Open `index.html` in your browser.
