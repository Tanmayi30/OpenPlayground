# BookBuddy

**Project Type:** Frontend (HTML/CSS/JS)
**Category:** Utility

## Description
BookBuddy is a book recommendation and reading log app for avid readers.

## Key Features
- Book recommendations by genre
- Personal reading log with local storage
- Responsive and accessible UI
- Search and filter log entries
- Export log to CSV
