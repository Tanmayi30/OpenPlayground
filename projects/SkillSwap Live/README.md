# SkillSwap Live

SkillSwap Live is a frontend peer-learning platform where users can host sessions, discover mentors, book slots, and exchange feedback.

## Features
- Create live skill sessions with tags, schedule, and seat limits
- Search sessions by skill tags
- Book sessions with duplicate-booking protection
- Toggle booking reminders
- Rate sessions and leave optional feedback comments
- Local persistence using browser `localStorage`

## Run
Open `index.html` in a browser.
