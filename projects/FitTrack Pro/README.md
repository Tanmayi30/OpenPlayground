# FitTrack Pro

FitTrack Pro is a frontend health and fitness dashboard to track daily workouts, calories burned, and progress against goals.

## Features
- Log workout type, duration, calories, and date
- Set daily targets for calories and workout minutes
- Visual progress cards for today's goals
- Last 7 days calorie trend chart
- Workout history list
- Export data as JSON
- Data persistence with browser localStorage

## Run
Open `index.html` in your browser.
