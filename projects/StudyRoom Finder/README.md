# StudyRoom Finder

StudyRoom Finder helps users discover nearby libraries and study spots with practical filters and quick shortlist tools.

## Features
- Search by location text and max distance
- Filters for noise level, Wi-Fi, sockets, and open-now status
- Save and manage favorite study spots
- Add crowd-sourced reviews and auto-update ratings
- Mobile-first responsive design
- Data persistence in browser localStorage

## Run
Open `index.html` in your browser.
